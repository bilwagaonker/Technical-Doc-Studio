# AI Technical Documentation Studio

An AI-powered platform that converts SAP demonstration videos into professional technical documentation.

## Vision

Automatically generate:
- Quick Reference Guides (QRGs)
- SAP training documentation
- Annotated screenshots
- DOCX, PDF, HTML, and Markdown outputs

using Computer Vision, OCR, Speech-to-Text, and Large Language Models.

## Planned Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- PostgreSQL

### AI
- OpenCV
- Whisper
- EasyOCR
- YOLO
- GPT-5.5 / Azure OpenAI
